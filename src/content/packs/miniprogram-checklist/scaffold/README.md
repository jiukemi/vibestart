# 微信小程序 · 待办清单

1. 打开**微信开发者工具** → 导入本项目目录
2. 确认 AppID（测试可用测试号）
3. 在 AI 编辑器中打开此目录，跟着分步提示词修改 `pages/index/` 下文件
4. 点击「预览」查看效果

## 文件说明

| 路径 | 作用 |
|------|------|
| `app.json` | 页面路由与窗口样式 |
| `pages/index/*` | 首页 WXML / JS / WXSS |
| `project.config.json` | 项目配置（compileType: miniprogram） |

需要云端同步时，可接入微信云开发（见向导「进阶后端辅助」）。
