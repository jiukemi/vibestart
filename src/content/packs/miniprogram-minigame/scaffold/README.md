# 微信小游戏 · 点击得分

1. 打开**微信开发者工具** → 导入本项目目录
2. 确认项目类型为**小游戏**（project.config.json 中 `compileType` 为 `game`）
3. 在 AI 编辑器中打开此目录，跟着分步提示词修改 `js/main.js`
4. 点击「预览」或「真机调试」查看效果

## 文件说明

| 文件 | 作用 |
|------|------|
| `game.js` | 小游戏入口 |
| `game.json` | 屏幕方向等配置 |
| `js/main.js` | Canvas 绘制与触摸逻辑（主要改这里） |

需要云端排行榜或存档时，可接入微信云开发（见向导「进阶后端辅助」）。
